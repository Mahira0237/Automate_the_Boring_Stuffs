1. What is a relative path relative to?
A relative path is relative to the program’s current working directory

2. What does an absolute path start with?
An absolute path always begins with the root folder

3. What does Path('C:/Users') / 'Al' evaluate to on Windows?
homeFolder / subFolder

4. What does 'C:/Users' / 'Al' evaluate to on Windows?

5. What do the os.getcwd() and os.chdir() functions do?
We can get the current working directory as a string value with the Path.cwd() function 
and change it using os.chdir()

6. What are the . and .. folders?
A single period (“dot”) for a folder name is shorthand for “this directory.” 
Two periods (“dot-dot”) means “the parent folder.

spam.txt base name

8. What are the three “mode” arguments that can be passed to the open() function?
read r, write w and append a

9. What happens if an existing file is opened in write mode?
Write mode will overwrite the existing file and start from scratch

10. What is the difference between the read() and readlines() methods?
the read() method returns the string that is stored in the file
and the readlines() method returns a list of string values from the file, one string for each line of text
